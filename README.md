# 智账

智账是一个面向大学生日常生活场景的本地桌面记账程序，使用 Python 和 tkinter 开发，支持多账本管理、收支录入、分类筛选、图表分析和 PDF 导出。项目强调轻量、易上手、可离线使用，适合作为课程设计、Python 桌面应用练习和个人记账工具。

## 项目特点

- 本地运行，无需联网，无需额外部署数据库
- 支持多账本切换，便于按月份、学期或用途分类管理
- 支持固定类别录入，减少自由输入带来的数据混乱
- 支持按类别筛选记录，便于快速查看支出结构
- 支持收支趋势图和分类饼图分析
- 支持一键导出 PDF 报表，方便汇报、打印和归档
- 支持 Windows 双击批处理脚本直接启动

## 技术栈

- Python 3.10+
- tkinter
- pandas
- matplotlib
- CSV 本地文件存储

## 功能说明

### 账本管理

程序支持新建、切换和删除账本。每个账本都会保存为 books 目录下的一个独立 CSV 文件。

### 收支记录

用户可以录入日期、类别、金额和备注。类别使用固定选项，包含餐饮、购物、交通、学习、娱乐、通讯、健康、社交、其他和收入。

### 数据展示与筛选

主界面使用表格显示当前账本中的全部记录，并支持按类别筛选以及批量删除所选记录。

### 图表分析

程序可根据当前账本数据绘制月度收支趋势图和分类饼图，用于观察消费变化和结构分布。

### PDF 导出

程序支持将统计表、收入饼图、支出饼图和趋势图导出为同一个 PDF 文件。

## 运行环境

建议使用 Python 虚拟环境运行本项目。

### 1. 创建虚拟环境

Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
```

macOS 或 Linux:

```bash
python -m venv .venv
source .venv/bin/activate
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

## 启动方式

### Windows

可直接双击 启动记账.bat。

### 命令行启动

```bash
python main.py
```

如果已经创建项目虚拟环境，也可以直接使用对应解释器启动。

## 数据存储说明

- 所有账本数据默认存放在项目根目录下的 books 文件夹中
- 程序首次启动时会自动创建 books 目录
- 如果项目根目录存在旧版 records.csv，程序会在首次启动时自动复制到 books/records.csv
- 每个账本文件均采用 CSV 格式，便于查看、备份和迁移
