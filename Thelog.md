# 变更日志

## 2026-03-19

- UI美化优化：应用莫兰迪配色体系，按钮改为圆角悬停反馈样式，统一输入框/下拉框/表格视觉，提升整体现代感与丝滑交互。
- 崩溃修复：修正 ModernButton 中与 Tkinter 内部变量冲突的问题（_w/_h），解决启动时报错 `_tkinter.TclError: invalid command name "104"`。
- 视觉与交互增强：新增 Windows 高 DPI 适配改善字体清晰度；按钮加入平滑颜色过渡动画；账本与核心操作区补充图标化文案与顶部标题区，降低简陋感与操作生硬感。
- 继续美化：界面升级为卡片化分区，记录表加入斑马纹与金额语义色（收入/支出），增加状态栏实时反馈并优化操作区布局，整体观感更完整。
- 五轮深度美化：新增账本概览统计卡片、实时关键字搜索与一键重置、表格列头点击排序、行悬停高亮、窗口淡入动效与快捷键（Ctrl+F/Ctrl+R/Ctrl+E），进一步提升现代感和操作流畅度。
- 比例可见性优化：主内容改为可滚动容器，账本区/新增记录区/底部操作区改为分行布局，避免窗口缩放和高 DPI 场景下按钮与信息被遮挡或裁剪。
- 下拉与图表继续美化：统一 Combobox 下拉弹窗配色/选中态/边框；重构趋势图与分类饼图为莫兰迪主题（柔和网格、数值标注、双环饼图与空数据提示），主界面与图表风格保持一致。
- 图表中心升级：新增图表年份下拉、图表中心弹窗（趋势图/分类图/导出趋势图），并在图表模块新增趋势图 PNG 导出能力，图表操作体验更完整。
- 图表观感修正：趋势图改为仅显示有数据月份，并将盈余/亏损区间分色填充，避免长时间零值与负值填充带来的异常视觉；饼图自动隐藏过小占比文本，减少标签拥挤。

## 2026-03-24

- 路径可移植性改造：将数据目录与账本文件路径统一为基于项目目录的相对结构（books 目录），移除对机器绝对路径写法的依赖，项目整体可直接迁移到其他电脑运行。
- 结构去重检查：确认根目录 `records.csv` 与 `books/records.csv` 存在历史兼容性重复；保留一次性迁移逻辑，仅将运行时读写固定到 `books` 目录。
- 细节修复：修正新建账本时可能重复写入 `books` 列表的问题，避免账本下拉中出现重复项。
- 图表文字修复：修正 Matplotlib 主题覆盖字体配置的顺序问题，增加系统字体文件自动加载策略，解决趋势图/饼图中文显示为方块的问题。
- 导出逻辑统一：重构 PDF 导出为“摘要页 + 新版趋势图 + 新版分类饼图”，导出内容与界面图表风格保持一致，不再回落旧版图表实现。
- 头部文案精简：移除主标题右侧副标题“极简记录，柔和配色”，仅保留 Smart Account 标题，界面更简洁。
- 收支统计修正：账本概览改为按类别判定，只有“收入”计入收入，其余类别全部计入支出，并将结余改为“收入 - 支出”。
- 表格对齐优化：记录表“金额/备注”列头锚点分别调整为右对齐/左对齐，与对应列数据对齐方式保持一致。
- 图表分组优化：分类饼图中的“收入构成”改为按备注分组统计，空备注自动归类为“未备注”，收入来源区分更清晰。
- 图表入口精简：移除主界面的“图表中心”按钮与对应弹窗逻辑，减少重复入口。
- 导出能力调整：图表区按钮由“导出趋势图”改为“导出饼图”，并新增分类饼图 PNG 导出方法，导出结果与当前饼图样式一致。
- 交互补充：恢复“导出趋势图”按钮并与“导出饼图”并存，分别支持趋势图（按年份）和饼图的一键 PNG 导出。
- 滚动行为修复：移除内容区的全局滚轮绑定（bind_all），改为仅外层内容画布响应，避免在记录表滚动时整页跟随联动滚动。
- 滚轮路由优化：恢复全局滚轮监听并增加事件路由判断，鼠标位于记录表区域时仅由表格处理滚动，其他区域可正常滚动外层页面。
- 布局顺序调整：将“删除选中/筛选/检索/重置”操作区移动到记录列表上方，查找和批量操作更贴近列表入口。
- 操作区美化：将记录列表上方操作区改为紧凑工具栏布局（删除区与筛选区同排分组、分隔线、统一留白与间距），解决控件拉得过开、视觉无序的问题。
- 首屏显示修复：记录表根据记录数自动调整可见行（10~14 行），并在渲染后触发布局刷新，避免默认窗口下需手动缩放后才能看到完整记录的问题。
- 记录列表宽度微调：仅增大“记录列表”卡片左右外边距（其余上方模块不变），使该模块视觉更收敛并留出两侧空白。
- 记录列表排版调整：外框恢复与上方模块同宽对齐；表格显示区与滚动区内缩至约 90%；“日期/类别/金额/备注”四列表头与内容统一居中。
- 窗口标题同步：主窗口左上角标题由“My记账”改为“Smart Account”，与界面左上主标题保持一致。
- 启动重置筛选：应用启动时自动清空“筛选类别”和“检索”输入框，防止遗留过滤条件影响初始数据展示。
- 数据容错：修复 `records.csv` 行末多余逗号导致读取列错位的问题，新增 `load_dataframe` 解析逻辑并把金额列异常数据强制为数字（错误值如“奶茶”将显示为0.00），防止启动时报 `ValueError` 崩溃。
- 趋势图修正：全量数据模式下按实际年月顺序绘制 X 轴（2025-09 -> 2026-03），避免跨年内容按“月份”错排。
- 下拉滚动修复：全局滚轮路由新增对 `ttk.Combobox` 下拉弹层（popdown Listbox）的识别，鼠标位于下拉列表时不再触发页面滚动，修复下拉窗口随滚轮错位问题。
- 全屏模式支持与布局稳定：新增 F11 切换全屏、Esc 退出全屏，并优化了图表区按钮容器，避免全屏下按钮排列紊乱。
- 下拉动画重构：统一账本/年份/日期/类别/筛选类别的下拉为同一动画组件；同一时刻仅允许一个下拉展开，打开新下拉会自动收回前一个；加速呼出与收回动画，并按莫兰迪主题统一下拉面板配色与选中态。
- 下拉收起增强：下拉展开后，点击任意其他控件、滚轮滑动到下拉外区域或应用失去焦点时，均自动收回下拉，避免弹层长期置顶。

- 文案微调：移除记录列表工具栏中“删除选中”按钮前的前导符号“-”，按钮文案改为“删除选中”，提升界面文案一致性与可读性。
